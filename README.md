# 엣지/우레탄 마감 일일 품질점검 현황

수동 입력과 엑셀 붙여넣기를 지원하고, Google Sheets 공유 저장소를 통해 여러 PC에서 동일한 데이터를 확인할 수 있는 품질점검 대시보드입니다.

## 링크

- 공개 대시보드: https://hyuk6188.github.io/packaging-line-quality-inspection/
- 공유 Google Sheet: https://docs.google.com/spreadsheets/d/1Egpz6f5SjwrwIIncoTzWA6Qsh1W5cx7oUF9OYYsiSXw/edit

## 주요 기능

- 설비별 금일, 월 누적, 연 누적 적합·부적합 현황
- 부적합 상세 내용과 검사 이미지 등록
- 수동 입력 및 엑셀 데이터 붙여넣기
- Google Sheets 자동 저장 및 30초 간격 동기화
- 대시보드 이미지 복사 및 인쇄

Apps Script를 다시 배포해야 하는 경우 `GOOGLE_SHEETS_SYNC_SETUP.md`를 참고하세요.
